
# Aula Markdown 

### Use # para criar titulos (use até 6 #)

# Titulo 1 
## Titulo 2 
### Titulo 3 
#### Titulo 4 
##### titulo 5
###### titulo 6

---

### Use sinal (-) ou (*) para criar tópicos 

* tópico (*)
- tópico (-)

---


### Use um (*) para deixar em italico e dois (**) para deixar em negrito, e use (~~) para taxado.

*Italico*

**Negrito**

~~Taxado~~

---
### Para criar listas numeradas use o número segudo de (.) 

1. item 
    1. item 
    1. item 
1. item 
1. item 
1. item
